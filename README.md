# Robot Learning Pages (GitHub Pages)

A minimal black & white learning site for "Autonomous Driving + Embodied Intelligence" study.

## Quick deploy (GitHub Pages)
1. Create a new public repo on GitHub, e.g. `robot-learning`.
2. Upload **all files** in this folder to the repo root.
3. Go to **Settings â†’ Pages**
   - Source: Deploy from a branch
   - Branch: `main` / folder: `/root`
4. Your site will be available at:
   - `https://<your-username>.github.io/robot-learning/`

## Update
Replace files in the repo and commit again. GitHub Pages will redeploy automatically.

## Notes
- Progress checkboxes are stored in browser `localStorage`.
- Formulas are rendered by MathJax (loaded from CDN).
